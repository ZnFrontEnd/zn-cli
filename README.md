# zn-cli

A light tool to generate projects in an easy way.

## 安装项目

```nginx
npm install zn-cli -g
```

## 初始化项目

```nginx
zn-cli init <template> <projectName>
```

tips：

* `<template>` : react/vue

* `<projectName>`: 项目的名称

## 启动项目

```
cd [projectName]
npm install
npm run dev
```

启动成功后，在浏览器中输入： `localhost:8080`即可访问
